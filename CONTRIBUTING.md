# Contributing

Thanks for contributing! This repo documents **real-world LLM gotchas** — the ugly truths that AI vendor marketing skips.

## How to Contribute

1. Fork this repo
2. Add your gotcha under the appropriate section (or create a new one)
3. Follow the format:
   - **Clear title** describing the gotcha
   - **What they tell you** — the marketing claim
   - **What actually happens** — the reality
   - **How to deal with it** — actionable advice
4. Submit a PR

## Guidelines

- Back claims with data, benchmarks, or real-world experience
- Be honest but fair — acknowledge when LLMs are genuinely impressive
- Include model names, versions, and dates where relevant
- Focus on production/enterprise use cases, not toy demos
